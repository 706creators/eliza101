4 常见问题解决

1. Install、Build 失败

  • 检查 NodeJS 版本
  • 检查 Command Line Tools
  • 检查 Git Tag

2. Build 总是失败

尝试清理后重新安装

3. 修改代码后不生效

  • 方案 1 因为,pnpm run start 执行的是 build 后的代码，所以，可以先 build。 pnpm run build && pnpm run start

  • 方案 2 使用 pnpm run dev 代替 pnpm run start
